[官网](https://b4a.clz.me)  
[开发板/类库/示例资源](https://github.com/coloz/b4a-cloud)  

## 开发  
### 关键技术  
Blockly 8  
Angular 13  
Electron 18  

### 安装依赖
```sh
npm i
cd app  
npm i
```
### 运行
```sh
npm start
```

### 编译
```sh
npm run electron:build
```
